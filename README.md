## Go 发布历史

- 2012-03 Go语言第一个正式版本Go1发布。
- 2015-08月，Go 1.5版发布(移除了”最后残余的C代码”)。
- 2018-08月，Go 1.11版

## Go 关键特性

- 并发与协程
- 基于消息传递的通信方式
- 丰富内置数据类型
- 函数多返回值
- defer机制
- 反射(reflect)
